# EcoMedellín - Landing Page Proyecto Ecológico
## Descripción del Proyecto

EcoMedellín es una landing page informativa diseñada para promover los espacios ecológicos, parques y jardines naturales de la ciudad de Medellín. El proyecto busca brindar información visual e interactiva sobre lugares ecológicos importantes y fomentar buenas prácticas ambientales mediante normas ecológicas.

---

## Tecnologías Utilizadas
* **HTML5**
* **CSS**
* **Font Awesome** (Iconografía)
* **Google Maps Embed API** (Mapas interactivos)

---

## Funcionalidades Implementadas

### Navegación Principal

* Barra de navegación sticky.
* Efectos hover interactivos.
* Enlaces de navegación interna mediante anclas.

### Hero Section

* Imagen de fondo responsive.
* Overlay oscuro para mejorar contraste.
* Botón CTA (Call To Action).

### Sección de Lugares Ecológicos

* Información detallada de parques y jardines.
* Galería de imágenes responsive con CSS Grid.
* Mapas interactivos embebidos con ubicación real.

### Seccion de Normas Ecológicas

* Cards informativas con iconografía.
* Diseño visual tipo cápsula.
* Hover animations.

### Footer

* Información de copyright.
* Diseño minimalista profesional.
---
## Conceptos Técnicos Aplicados

### CSS Grid

Se utilizó para estructurar:

* Layout principal de contenido.
* Galerías de imágenes.
* Distribución de cards de normas.

### Flexbox

Se utilizó para:

* Navbar.
* Alineación interna de cards.
* Centrado de contenido.

### Variables CSS

Se implementó `:root` para manejo centralizado de colores.

### Responsive Design

Uso de media queries para adaptación móvil/tablet.
